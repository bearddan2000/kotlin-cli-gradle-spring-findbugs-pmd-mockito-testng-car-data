# kotlin-cli-gradle-spring-findbugs-pmd-mockito-testng-car-data

## Description
A POC for spring app using testng,
pmd, mockito framework.

## Tech stack
- kotlin
- gradle
  - spring
  - testng
  - mockito
  - pmd
  - findbugs

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
